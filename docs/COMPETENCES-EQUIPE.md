## 4. Plan de Formation

### Action de Formation Proposée : Tests Automatisés

**Objectif** : Acquérir les compétences en tests unitaires et d'intégration

**Format** : Auto-formation + pratique sur le projet

**Durée** : 2 semaines

**Programme** :

| Semaine | Contenu | Livrables |
|---------|---------|-----------|
| S1 | Fondamentaux Vitest + Testing Library | 3 tests unitaires composants React |
| S2 | Tests API + Mocking | 3 tests endpoints backend |

**Ressources** :

| Ressource | Type | URL |
|-----------|------|-----|
| Vitest Documentation | Officielle | https://vitest.dev |
| Testing Library | Officielle | https://testing-library.com/docs/react-testing-library/intro |
| Kent C. Dodds Blog | Articles | https://kentcdodds.com/blog |
| Fireship Testing | Vidéo | YouTube |

**Critères de Réussite** :

- [ ] 6 tests écrits et passants
- [ ] Couverture > 20% sur les fichiers critiques
- [ ] Tests exécutés dans le pipeline CI
- [ ] Documentation des tests créée
